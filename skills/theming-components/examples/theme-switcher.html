Design Tokens - Theme Switcher Demo

Welcome to Design Tokens

This demo shows how design tokens enable automatic theme switching. All visual styling comes from CSS custom properties (design tokens).

Try switching themes above - notice how all components update instantly without any JavaScript manipulation of styles.

How It Works

  1. Set data-theme attribute on <html>
  2. CSS overrides token values for that theme
  3. All components referencing tokens update automatically
