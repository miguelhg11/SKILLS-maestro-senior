# Palo Alto Networks Security Dashboard

A security analytics dashboard built using the **AI Design Components Skill Chain**.

## Skills Used

This dashboard was generated by chaining the following skills:

| Skill | Purpose |
|-------|---------|
| `theming-components` | Brand colors (#FA582D Cyber Orange), design tokens, light/dark themes |
| `designing-layouts` | Responsive grid (3 → 2 → 1 columns), header + cards layout |
| `creating-dashboards` | KPI card patterns, dashboard structure |
| `visualizing-data` | Donut chart for attack source breakdown |
| `providing-feedback` | Toast notifications, spinner loading, empty states |

## Components

### KPICard
Displays key performance indicators with trend indicators and severity badges.

```tsx
<KPICard
  label="Threats Blocked"
  value="12,847"
  trend={15.3}
  trendLabel="vs last month"
  severity="info"
/>
```

### DonutChart
Displays composition data with interactive segments and legend.

```tsx
<DonutChart
  title="Attack Sources"
  data={[
    { label: 'External', value: 4521 },
    { label: 'Phishing', value: 3892 },
    { label: 'Malware', value: 2134 },
  ]}
/>
```

### Toast Notifications
Global notification system with success, error, warning, and info variants.

```tsx
const { success, error } = useToast();
success('Saved!', 'Your changes have been saved.');
```

### EmptyState
Displays when no data is available or an error occurred.

```tsx
<EmptyState
  title="No threats detected"
  description="Your network is secure."
  variant="default"
/>
```

## Theme Switching

The dashboard supports light and dark themes via CSS custom properties:

```javascript
// Switch to dark mode
document.documentElement.setAttribute('data-theme', 'dark');

// Switch to light mode
document.documentElement.setAttribute('data-theme', 'light');
```

## Design Tokens

All styling uses CSS custom properties from `tokens.css`:

- **Colors**: `--color-cyber-orange`, `--color-prisma-blue`, etc.
- **Spacing**: `--spacing-sm`, `--spacing-md`, `--spacing-lg`
- **Typography**: `--font-size-base`, `--font-weight-semibold`
- **Shadows**: `--shadow-sm`, `--shadow-md`, `--shadow-lg`
- **Motion**: `--duration-fast`, `--transition-normal`
